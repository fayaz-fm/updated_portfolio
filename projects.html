Height-Based Object Sorting

Aim: Automatically sort objects based on height using industrial automation.
Working: Objects are placed on a conveyor, where a height sensor detects their size. The signal is processed by the PLC (Allen-Bradley Micro850), which activates diverters to sort items. Taller items go one direction, shorter items go another.
Components: Micro850 PLC, Factory I/O, height sensors, conveyor belts, actuators.
Tools: Connected Components Workbench (CCW), Factory I/O.

Water Tank Automation

Aim: Automate tank water level control.
Working: PLC timers manage pump operation based on level sensors. Ignition SCADA provides visual monitoring and control.
Components: Micro850 PLC, tank, pump, level sensors.
Tools: CCW, Ignition SCADA.

Obstacle Detection & Monitoring

Aim: Detect obstacles and measure distance in real-time.
Working: Ultrasonic sensor sends distance data to Arduino which processes and alerts on proximity.
Components: Arduino, ultrasonic sensor, LCD, LED Lights
Tools: Arduino IDE.

Smart Robot Car 4.0

Aim: Create a mobile-controlled robot car with obstacle avoidance.
Working: Bluetooth module receives commands from mobile app. Ultrasonic sensors detect obstacles. Microcontroller drives motor accordingly.
Components: Arduino, motor driver, ultrasonic sensor, Bluetooth module, wheels.
Tools: Arduino IDE, MIT App Inventor.

Bluetooth-Controlled DC Motor

Aim: Control DC motor speed via Bluetooth.
Working: MSP430 receives speed commands and controls motor. LCD displays real-time speed.
Components: MSP430, Bluetooth module, LCD, DC motor.
Tools: Code Composer Studio, Energia.

Beverage Filling Automation

Aim: Automate beverage filling process.
Working: Sensors detect bottle, timers activate filling and discharge. PLC logic synchronizes stages.
Components: Sensors, TLP Logix Pro, filling mechanism.
Tools:TLP LogixPro

Pick-and-Place Robot – ABB

Aim: Simulate pick-and-place using ABB RobotStudio.
Working: Robot program in RAPID controls arm to pick and place components based on coordinates.
Components: ABB IRB120 model, gripper tool.
Tools: ABB RobotStudio, RAPID language.
